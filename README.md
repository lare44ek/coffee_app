# Приложение для учёта продукции в кофейне

---

## Описание

`Effoc app` — это мобильное приложение, разрабатываемое для упрощения и автоматизации учёта продукции в кофейне. Основная цель приложения — генерировать маркировки с информацией о дате вскрытия и сроке годности продукта, а также отслеживать историю этих маркировок. Это помогает персоналу кофейни эффективно управлять свежестью ингредиентов.

---

## Цели

*   **Упрощение маркировки:** Автоматически рассчитывать и формировать этикетки для вскрываемых продуктов с указанием даты и времени вскрытия и окончания срока годности.
*   **Отслеживание:** Хранить историю сгенерированных и, потенциально, отсканированных маркировок.
*   **Удобство использования:** Предоставить интуитивно понятный интерфейс для персонала кофейни.
*   **Производительность:** Будет работать даже на курилке

---

## v0.1 alpha

Пока реализованы следующие функции:

*   **Авторизация:** Простой экран логина для входа в приложение.
*   **Генератор маркировок:** Выбор продукта из предустановленного списка, расчёт даты/времени окончания срока годности и отображение готовой маркировки.
*   **Сканирование QR-кода:** Экран с доступом к камере для сканирования QR-кодов (функционал распознавания пока не реализован, но экран интегрирован).
*   **История маркировок:** Возможность сохранять сгенерированные маркировки и просматривать их список на отдельном экране.
